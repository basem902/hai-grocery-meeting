# ورقة اجتماع — سلسلة بقالات عبدالحافظ

صفحة HTML واحدة، عربية (RTL)، محسّنة للجوال، **مرتّبة بترتيب الاجتماع** — تُستخدم أثناء الجلسة لمناقشة كيف نحمي سلة الحي اليومية قبل أن تنتقل للتطبيقات (كيتا / نون مينتس / هنغرستيشن).

## الاستخدام
- افتح **`index.html`** في متصفح الجوال — لا تحتاج إنترنت لتعمل (الخطوط فقط تُحمّل أونلاين، ويوجد بديل من خط النظام).
- انزل بالصفحة خطوة بخطوة؛ الأقسام مرتّبة كترتيب الاجتماع نفسه.

## المحتوى (10 خطوات + المصادر)
1. القاعدة الذهبية (اسمع 70٪ · تكلّم 30٪)
2. النبرة الصحيحة + جملة الارتكاز
3. الصورة الحقيقية للسوق — أرقام مفروزة (🟢 سعودي موثّق / 🟡 تشبيه عالمي)
4. سلاحك: المنظومة الكاملة (عميل · سائق · ولاء · مخزون · تقارير · موزّع)
5. أسئلة سحب الألم (مرتّبة ومرقّمة)
6. **حاسبة العميل المتسرّب — حيّة ومدمجة** (عدّل أرقامه أمامه فتظهر الخسارة فوراً)
7. ربط كل ألم بحل جاهز
8. تجربة الـ30 يوم
9. اعتراضات جاهزة + ردودها
10. جمل حاسمة + قل / لا تقل

## ملاحظات مهمة
- **الأرقام ذخيرة نقاش، لا تقرير تدقيق رسمي.** الأخضر = موثّق سعودي يُقال بثقة؛ الأصفر = تشبيه عالمي يُقدَّم كـ«في أسواق مشابهة».
- تصحيح مهم: **نانا (Nana)** هي التي دخلت إعادة هيكلة — **لا نون (noon)**.
- الحاسبة تعمل بالكامل في المتصفح (JavaScript)، بلا أي اعتماد خارجي.
